# Enterprise Multi-Document RAG Chatbot

AI-powered Retrieval-Augmented Generation platform built using:

- Next.js 14
- FastAPI
- PostgreSQL
- Redis
- FAISS
- BM25 Hybrid Search
- Gemini AI

## Features

- Multi-document ingestion
- Semantic search
- Hybrid Retrieval (FAISS + BM25)
- Role-based access
- Multi-tenant architecture
- Modern responsive UI

## Tech Stack

Frontend:
- Next.js
- TypeScript
- Tailwind CSS
- Zustand

Backend:
- FastAPI
- SQLAlchemy

Database:
- PostgreSQL

Caching:
- Redis

AI:
- Gemini
- FAISS
- BM25
